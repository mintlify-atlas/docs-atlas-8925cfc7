# Welcome to Wonderous

Wonderous is a beautifully crafted Flutter application that takes you on a journey through history, art, and culture. Explore some of the most famous structures in the world while experiencing the full potential of Flutter's visual capabilities on modern mobile hardware.

<Note>
  Built by [gskinner](https://gskinner.com/) in partnership with the Flutter team, Wonderous deliberately pushes visual fidelity, effects, and transitions to showcase what Flutter is truly capable of.
</Note>

## Key Features

### Rich Visual Experience
Wonderous leverages Flutter's advanced rendering capabilities to deliver:
- Smooth animations and transitions
- High-fidelity visual effects
- Impeller rendering on iOS for enhanced performance
- Responsive design across mobile, tablet, and desktop

### Educational Content
- Detailed historical information about each wonder
- Interactive timeline spanning from 3000 BCE to 2200 CE
- Photo galleries with high-quality imagery
- Artifact search powered by museum APIs
- Editorial content with engaging storytelling

### Modern Architecture
The app demonstrates Flutter best practices:
- **State Management**: Uses `GetIt` for dependency injection and `Provider` for state management
- **Routing**: Implements `go_router` for declarative navigation
- **Internationalization**: Supports multiple languages via Flutter's i18n system
- **Multi-platform**: Runs on iOS, Android, Web (WASM), macOS, and Windows

### Interactive Features
- Collectibles system with discovery mechanics
- Home widgets for quick access to wonder information
- Photo galleries integrated with Unsplash
- Interactive maps using Google Maps
- YouTube integration for video content
